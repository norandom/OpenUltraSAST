# OpenUltraSAST

OpenUltraSAST is an independent OpenCode security harness project. It combines HarnessX-style harness composition, OpenUltraCode verification discipline, OpenRouter-hosted models and embeddings, Docker-isolated code analysis, and Trail of Bits security skills into a SAST-oriented workflow.

The initial project spec lives in:

- `.kiro/specs/openrouter-sast-harness/requirements.md`
- `.kiro/specs/openrouter-sast-harness/design.md`
- `.kiro/specs/openrouter-sast-harness/tasks.md`
- `.kiro/specs/openrouter-sast-harness/assessment.md`

Clearwing is used as an implementation oracle for proven source-hunting ideas, not as a dependency or fork target.
